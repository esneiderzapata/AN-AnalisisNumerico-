Gauss-Seidel

 Realiza el metodo de Gauss-Seidel para sistemas Ax=b utilizando la matriz A y el vector B, mientras A sea estrictamente diagonal dominante

Para ingresar la matriz A, introduce los valores de las filas separados por espacios, y separa las columnas con #
Por ejemplo: 1 34 2 # 8 5 2 # 1 2 3
Da como resultado la matriz:
([1, 34, 2],
[8, 5, 2],
[1, 2, 3])

Para introducir el vector B lo hacemos en modo de columna, es decir: 1 # 2 # 3
([1],
[2],
[3])

El vector x0, se introduce unicamente con espacios: 0 0 0
